# Trabalho de Organização de Computadores

## Fazer um simulador RISC-V

# Para rodar o código

Para rodar o código, basta executar um dos comandos:

`docker compose up`, ou
`podman-compose up`, ou
`python risc-v.py`

# Para rodar todos os testes

Para rodar todos os testes do unittest basta colar o comando:

`python -m unittest discover`
